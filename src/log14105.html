go-eCharger (14105)

Der Baustein dient zur Kommunikation mit einer go-e Ladestation / Wallbox, s. API.

Eingänge

Eingang Initwert Beschreibung
IP IP der Wallbox
Port 0 Port der Wallbox
Intervall 0 Bei einem Wert <> 0 werden die Daten go-e Wallbox zyklisch mit dem angegebenen Intervall in Sekunden abgerufen.
Trigger 0 Bei einem Wert <> 0 werden die Daten go-e Wallbox abgerufen.
Für die weiteren Parameter, s. go-eCharger API. Von der API abweichende Einheiten sind am jew. Ein- Ausgang des Bausteins angegeben.

Ausgänge

Ausgang Initwert Beschreibung
Online 0 1, wenn Wallbox erreichbar; 0, wenn nicht
Für die Parameter, s. go-eCharger API

Sonstiges

Neuberechnung beim Start Nein
Baustein ist remanent Nein
